# Szereg aktywności metali

97/294

K>Ca>Na>Mg>Al>Zn>Pb>Cu>Ag

101/295

CuSO₄ + Ag → x

CuSO₄ + Zn → ZnSO₄ + Cu

CuSO₄ + Cu → x

CuSO₄ + Na → Na₂SO₄ + Cu

102/295

Zn + FeSO₄ → ZnSO₄ + Fe Zn>Fe

Mg + FeCl₂ → MgCl₂ + Fe Mg>Fe

Cu + ZnCl₂ → x Cu<Zn

**Półogniwo** - składa się z blaszki metalu zanurzonej w roztworze dobrze rozpuszczalnej soli tego metalu

np. półogniwo miedziane: Cu/Cu^2+ lub Cu w CuSO₄

półogniwo cynkowe: Zn/Zn^2+ lub Zn w ZnSO₄
