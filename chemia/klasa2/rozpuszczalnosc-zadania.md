### 110/296

| Dane:                | Szukane:  |
| -------------------- | --------- |
| $`m_r = 180g`$       | $`R = ?`$ |
| $`m_s = 60g`$        |
| $`T = 20 \degree C`$ |

$`m_r = m_s + m_{H_2O}`$

$`180g = 60g + m_{H_2O}`$

$`m_{H_2O} = 180g - 60g = 120g`$

$`\frac{60g}{x} \cdot \frac{120g}{100} = 120x = 6000g = x = 50g`$

$`R = 50g/100g_{H_2O}`$
